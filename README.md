# PM & TPM Playbooks

A living collection of frameworks, templates, and structured thinking 
built from real product and program work across Ad Tech, AI/ML, and 
Marketplace products.

Written from the perspective of someone who has shipped LLM-powered 
products, measurement frameworks, and large-scale ad format launches 
at a top-tier tech company.

## What's inside

| Area | Description |
|------|-------------|
| 📊 Measurement | Privacy-aware measurement, signal loss, attribution |
| 🤖 LLM Products | Eval frameworks, model-human parity, agentic design |
| 🏪 Marketplace | RCA playbooks, anomaly detection, quality metrics |
| 🧪 Experimentation | A/B design, guardrail metrics, rollout decisions |
| 📋 PM/TPM Templates | PRDs, OKRs, stakeholder maps, program reviews |
| 🎯 Strategy | Product thinking, prioritisation, roadmap frameworks |

## How to use this
Each folder contains standalone markdown documents — frameworks you 
can adapt, templates you can reuse, and thinking you can build on.

## Status
🚧 Actively building — added as frameworks are developed and refined.
